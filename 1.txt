3)3. Jenkins Freestyle Job with String Parameter

Goal: When you build the job with a parameter STUDENT_NAME, it prints:

Hello STUDENT_NAME, Welcome to Jenkins!

(but with the actual name substituted)

Step-by-step (Linux shell build step)

Open Jenkins dashboard.

Click New Item.

Enter a name, e.g. Hello-Student-Job.

Select Freestyle project ‚Üí click OK.

On the job config page:

Tick This project is parameterized.

Click Add Parameter ‚Üí choose String Parameter.

Fill:

Name: STUDENT_NAME

Default Value: e.g. Omprasad

(Optional) Description: Enter your name

Scroll to Build section ‚Üí click Add build step ‚Üí

In Build ‚Üí Add build step ‚Üí Execute Windows batch command:
In the command box, put:
In Build ‚Üí Add build step ‚Üí Execute Windows batch command:

echo Hello %STUDENT_NAME%, Welcome to Jenkins!





Click Save.

To run it:

Open the job ‚Üí click Build with Parameters.

Enter a value for STUDENT_NAME (e.g., Omprasad).

Click Build.

Open the build ‚Üí Console Output ‚Üí you should see:

Hello Omprasad, Welcome to Jenkins!
